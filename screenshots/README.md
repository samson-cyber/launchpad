# Screenshot Guide

Chrome Web Store requires 1-5 screenshots at **1280x800** pixels (or 640x400 minimum).
Use PNG format for crisp text.

## How to capture at 1280x800

1. Open Chrome DevTools (F12)
2. Click the **device toolbar** icon (or Ctrl+Shift+M)
3. Set dimensions to **1280 x 800**
4. Navigate to the new tab page
5. Take a screenshot: DevTools menu (⋮) → "Capture screenshot"

## Screenshots to capture

### screenshot-1-light.png — Main view (light mode)
Set up this state before capturing:
- 3-4 groups: "Work", "Social", "Dev Tools", "News"
- 4-6 shortcuts per group (use popular sites: Google, GitHub, YouTube, Reddit, Twitter, Stack Overflow, Gmail, Notion, Figma, etc.)
- Light mode active
- Search bar visible at top
- Settings gear and theme toggle visible in corners

### screenshot-2-dark.png — Main view (dark mode)
- Same shortcuts and groups as screenshot 1
- Toggle dark mode (click the moon icon)
- Shows the dark #202124 background with light text

### screenshot-3-add-shortcut.png — Add Shortcut modal
- Open the "Add shortcut" modal (click + on any group)
- Type a URL like "https://github.com" in the URL field
- Name field auto-populated with "github.com"
- The modal should be visible over the main page (slightly dimmed)

### screenshot-4-import.png — Bookmarks import
- Open the bookmark import modal (gear icon → Import Bookmarks)
- Show several bookmark folders with checkboxes
- Check 2-3 folders to show the selection state
- Import button visible at bottom

## Tips
- Use a clean Chrome profile with no other extensions visible
- Hide the bookmarks bar (Ctrl+Shift+B) for a cleaner look
- Use real site names and URLs for authenticity
- Make sure favicons are loaded (requires internet connection)
