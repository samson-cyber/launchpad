# LaunchPad

Chrome's new tab page, but with unlimited organized shortcuts. No bloat.

## Features

- Unlimited shortcuts organized into custom groups
- Clean, fast new tab page replacement
- Fully local — no accounts, no cloud sync, no tracking
- Lightweight and distraction-free

## Installation

1. Clone or download this repository
2. Open Chrome and navigate to `chrome://extensions`
3. Enable **Developer mode** (toggle in the top right)
4. Click **Load unpacked** and select the project folder
5. Open a new tab to see LaunchPad

## Usage

Open a new tab and LaunchPad replaces the default Chrome page. Add, organize, and access your shortcuts instantly.

## Privacy

LaunchPad collects no data. Everything is stored locally on your device. No analytics, no tracking, no third-party services. See [privacy-policy.html](privacy-policy.html) for details.
